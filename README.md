# 공용 모듈 임시 / 스토리북

공용 컴포넌트를 위한 임시 스토리북

# 리스트

파일업로더

# 배포 방법

1. 빌드 파일 생성

아래 커멘드를 실행하여 빌드 파일 생성

```
npm run build-storybook
```

2. github 페이지에 정적 배포
   아래 커멘드를 실행하여 빌드 파일 생성

```
npm run storybook:deploy
```

# 로컬에서 실행 방법

아래 커멘드를 실행하여 로컬 스토리북 서버 실행

```
npm run storybook
```
